node 最流行的框架之一 express，官方介绍是基于 nodejs，快速、开放、极简的 web 开放框架。

**安装**

```shell
npm install express --save
```

使用 nodemon 来启动 express，避免重复启动。

```shell
npm i nodemon -g
```
